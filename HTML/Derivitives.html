Basic Derivitive Rules

  • Constant Rule: The derivative of a constant function is always zero. For example, if f(x) = c, where c is a constant, then f'(x) = 0.

  • Power Rule: The power rule states that if f(x) = x^n, where n is a constant,
    then f'(x) = nx^(n-1). This rule applies to functions with a single variable raised to a constant power.

  • Sum/Difference Rule: The derivative of the sum or difference of two functions is equal
    to the sum or difference of their derivatives. If f(x) = g(x) ± h(x), then f'(x) = g'(x) ± h'(x).

  • Product Rule: The product rule is used when we need to find the derivative of a function
    that is the product of two other functions. If f(x) = g(x) * h(x), then f'(x) = g'(x) * h(x) + g(x) * h'(x).

  • Quotient Rule: The quotient rule is used to find the derivative of a function that is the
    ratio of two other functions. If f(x) = g(x) / h(x), where h(x) ≠ 0, then f'(x) = (g'(x) * h(x) - g(x) * h'(x)) / (h(x))^2.

  • Exponential and Logarithmic Rules: The derivatives of exponential and logarithmic functions have
    specific rules. For example, the derivative of e^x is e^x, and the derivative of ln(x) (natural logarithm) is 1/x.

    • Chain Rule

    Chain Rule: The chain rule is applied when we have a composite function, where one function is
    applied as the input of another function. If f(x) = g(h(x)), then f'(x) = g'(h(x)) * h'(x).
    This rule is used to find the derivative of nested functions.

    Tangent Line Equation

  • Find the derivative of the function f(x) with respect to x, denoted as f'(x).
    This derivative represents the slope of the curve at any given point.

  • Substitute the x-coordinate of the point of tangency (x₀) into the derivative f'(x)
    to find the slope of the curve at that point. Let's denote this slope as m.

  • Now we have the slope (m) and a point (x₀, y₀) on the tangent line.
    We can use the point-slope form of a linear equation to describe the tangent line:

  • y - y₀ = m(x - x₀)

  • This equation represents the equation of the tangent to the curve at the point (x₀, y₀).
    It states that the difference in y-coordinates (y - y₀) is equal to the slope (m) times the difference in x-coordinates (x - x₀).

  • If desired, you can rearrange the equation to the slope-intercept form (y = mx + b) by simplifying and solving for y:

  • y = mx + (y₀ - mx₀)

  • In this form, the equation represents the tangent line in terms of the slope (m) and the y-intercept (b = y₀ - mx₀).

    • Implicit diferentiation

  • Start with an equation that relates x and y. For example, let's consider the equation: F(x, y) = 0

  • Assume that both x and y are functions of the independent variable, often denoted as t.
    So, we can express x and y as x(t) and y(t), respectively.

  • Apply the chain rule to differentiate both sides of the equation with respect to t.
    This means that for any function f(u), if u is a function of t, then the derivative of f(u)
    with respect to t is given by f'(u) * u'.

  • For example, differentiating x(t) with respect to t gives x'(t), and differentiating y(t)
    with respect to t gives y'(t).

  • Differentiate each term of the equation separately, using the chain rule whenever necessary.

  • For example, if we have the term x^2, differentiating it with respect to t gives 2x * x'(t).
    Similarly, if we have the term y^3, differentiating it with respect to t gives 3y^2 * y'(t).

  • After differentiating each term, rewrite the equation in terms of x, y, x', and y' only.
    Eliminate the derivative of the independent variable (t) from the equation.

  • Solve the resulting equation for y' to find the derivative of y with respect to x.

  • Note that y' represents the derivative of y with respect to the independent variable (t).
    However, once you solve the equation, you'll have an expression for y' in terms of x, y, and x'.

    • derivitives of inverse functions

  • The derivatives of inverse functions are related to the derivatives of the original functions through a specific formula.
    Let's say we have a function f(x) and its inverse function is denoted by f^(-1)(x).

  • If f(x) is differentiable at a point x=a and f'(a) ≠ 0, then the inverse function f^(-1)(x) is differentiable at x=b=f(a),
    and its derivative at that point is given by: (f^(-1))'(b) = 1 / f'(a)

  • In other words, the derivative of the inverse function at a particular point is equal to the reciprocal of the
    derivative of the original function evaluated at the corresponding point.

  • To illustrate this concept, let's consider an example. Suppose we have the function f(x) = 2x + 3,
    and we want to find the derivative of its inverse function at a particular point.

  • Step 1: Find the derivative of the original function f(x): f'(x) = 2

  • Step 2: Find the point where we want to evaluate the derivative of the inverse function, let's say x = 5.

  • Step 3: Evaluate f'(x) at x = 5: f'(5) = 2

  • Step 4: Find the corresponding value of f(x) at x = 5: f(5) = 2(5) + 3 = 13

  • Step 5: Use the formula to find the derivative of the inverse function at x = 13: (f^(-1))'(13) = 1 / f'(5) = 1 / 2 = 0.5

  • So, the derivative of the inverse function of f(x) = 2x + 3, evaluated at x = 13, is 0.5.

    • MVT

  • the Mean Value Theorem states that if a function f(x) is continuous on the closed interval [a, b] and differentiable on
    the open interval (a, b), then there exists at least one value c in the interval (a, b) such that the derivative of f(x) at c
    is equal to the average rate of change of f(x) over the interval [a, b]. In other words, if f(a) and f(b) are the values of the
    function at the endpoints of the interval, then there exists a point c where the instantaneous rate of change of f(x) at c is equal
    to the slope of the secant line connecting (a, f(a)) and (b, f(b)).

  • The Mean Value Theorem can be expressed mathematically as:

  • f'(c) = (f(b) - f(a))/(b - a)
    where f'(c) denotes the derivative of f(x) at c.

    • Function Analysis w/ Concavity

  • Find the second derivative: Differentiate the function f(x) once to obtain its first derivative f'(x), and then differentiate
    f'(x) again to obtain the second derivative f''(x).

  • Identify critical points: Determine the points where the second derivative is equal to zero or undefined. These points are known
    as critical points and can help us identify potential inflection points.

  • Test intervals: Divide the domain of the function into intervals using the critical points. For each interval, select a test point and
    evaluate the sign of the second derivative at that point.

  • a. If the second derivative is positive (+) at the test point, the function is concave upward on that interval.

  • b. If the second derivative is negative (-) at the test point, the function is concave downward on that interval.

  • Note: The second derivative being zero at a test point indicates a potential point of inflection, which requires further investigation.

  • Plotting the results: Based on the signs of the second derivative in different intervals, we can sketch the graph of the function.
  • The concave upward regions correspond to U-shaped curves, while the concave downward regions correspond to inverted U-shaped curves.

  • Points of inflection: If the second derivative changes sign at a specific point within an interval, it is a potential point of inflection.
    To confirm it as a point of inflection, analyze the behavior of the function at that point (e.g., using the first derivative test or higher-order derivatives).

    • Absolute Extrema

  • Identify the interval: Determine the interval or domain over which you want to find the extrema. It could be a closed interval [a, b] or an open interval (a, b).

  • Find critical points: Critical points are the locations where the derivative of the function is zero or undefined. To find critical points, take the derivative of the function
    and set it equal to zero, then solve for the variables. Also, consider any points where the derivative is undefined, such as vertical asymptotes.

  • Evaluate the function at critical points: Once you have the critical points, evaluate the function at each of these points and record the corresponding function values.

  • Evaluate the function at the endpoints: Evaluate the function at the endpoints of the given interval.

  • Compare the function values: Compare all the function values obtained from steps 3 and 4. The highest function value represents the absolute maximum, and the lowest function value
    represents the absolute minimum over the given interval.
